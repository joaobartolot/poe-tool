# Faustus Currency Calculator

A mobile-first calculator for Path of Exile Faustus currency trades.

## Features

- Dynamic calculation with no submit button.
- Handles decimal ratios and converts them into the largest whole-number trade.
- Shows exactly what to type in **I want** and **I have**.
- Includes leftover inventory and normalized ratio details.
- `npm run dev` starts Vite on `0.0.0.0` and prints a QR code for quick phone access.

## Tech

- React + Vite + TypeScript
- Tailwind CSS
- ESLint + Prettier (+ Tailwind class sorting)

## Commands

```bash
npm install
npm run dev
npm run lint
npm run build
npm run format:write
```
